    ```typescript
    export interface BookSpineProps {
      id?: string;

      author: string;

      onTap?: () => void;
    }
    ```

    2. A pure function `getContrastColor(hex: string): '#ffffff' | '#000000'` that:
       - Parses the hex string (strip leading `#`)
       - Extracts R, G, B as integers (0–255)
       - Computes luminance: `(R * 0.299 + G * 0.587 + B * 0.114) / 255`
       - Returns `'#ffffff'` if luminance < 0.5, else `'#000000'`
       - Handles edge cases: if hex is empty or unparseable, return `'#000000'`

    3. A default-exported `BookSpine` React component:

    **Dimensions (per UI-SPEC):**
    - Width: 56px fixed (targets 6–8 spines per row at 1280px viewport)
    - Height: 150px (fits title + author without truncation at typical lengths)

    **Layout (outer element):**
    - `<div>` with `role="listitem"` and `aria-label={title}`
    - Tailwind classes: `relative flex-shrink-0 border-2 border-black cursor-pointer`
    - Width and height via inline style: `style={{ width: '56px', height: '150px', backgroundColor: coverColor }}`
    - onClick handler: `onClick={onTap}` (no-op if undefined)

    **Text layout (inner):**
    - Single inner `<div>` absolutely positioned, filling the spine: `className="absolute inset-0 flex flex-col items-center justify-center overflow-hidden"`
    - Use CSS `writing-mode: vertical-rl` and `transform: rotate(180deg)` on a wrapper to rotate text 90° so it reads top-to-bottom down the spine
    - Inner wrapper div: `style={{ writingMode: 'vertical-rl', transform: 'rotate(180deg)', display: 'flex', flexDirection: 'column', alignItems: 'center', gap: '4px', maxHeight: '140px', overflow: 'hidden' }}`

    - Content: `{author}`
