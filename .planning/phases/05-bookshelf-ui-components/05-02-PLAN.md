    **Bookshelf component logic:**
    The component receives a flat `books` array and renders all spines in a single flex-wrap container. The "wrapping to a new shelf row" visual is achieved by the shelf background repeating — the container is styled as a flex-wrap row with the wood gradient and shelf lip, so when spines wrap they naturally continue on a visually contiguous background.

    **Outer container (semantic):**
    - `<section role="region" aria-label="Bookshelf" className="w-full">`

    **Shelf strip (the wooden surface):**
    A single `<div>` that contains all spines. Spines flex-wrap horizontally:
    ```
    className="flex flex-row flex-wrap items-end gap-0"
    style={{
      background: 'linear-gradient(to bottom, #8B6F47 0%, #6B5438 50%, #4A3728 100%)',
      borderBottom: '14px solid #4A3728',
      padding: '8px 8px 0 8px',
      rowGap: '8px',
      minHeight: '166px',   // 150px spine height + 8px top padding + 8px bottom gap
    }}
    ```

    **Rendering spines:**
    Map over `books` and render `<BookSpine key={book.id ?? String(index)} {...book} />` for each.

    **Empty state:**
    If `books.length === 0`, render inside the shelf strip:
    ```tsx
    <div className="w-full flex flex-col items-center justify-center py-xl">
      <p className="text-body font-georgia text-text-secondary">Add your first book</p>
      <p className="text-subtitle font-georgia text-text-secondary mt-sm">No books yet. Tap the + button to add one.</p>
    </div>
    ```

    Import `BookSpine` as default import: `import BookSpine from './BookSpine'`
    Export `Bookshelf` as both named and default export for flexibility:
    ```typescript
    export { Bookshelf }
    export default Bookshelf
    ```
