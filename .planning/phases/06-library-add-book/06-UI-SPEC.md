## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, rotated text gap on book spine |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing, form field padding, list gaps |
| lg | 24px | Section padding, form section breaks |
| xl | 32px | Layout gaps, page content width padding |
| 2xl | 48px | Major section breaks, header/footer spacing |
| 3xl | 64px | Page-level spacing |

Exceptions: None. Book spine dimensions (56px × 150px) are fixed absolute measurements, not part of this scale.

## Component Specifications

### "+" Button (Add Book Entry Point)

**Location:** Top-right of `/library` page header

**Dimensions:** 48px × 48px

**Styling:**
- Background: aged gold (#D4AF37)
- Border: 2px solid black (#000000)
- Content: "+" symbol, centered, 24px font size, charcoal text (#2C2C2C)
- Cursor: pointer
- Hover state: opacity 0.9 (subtle darken)
- Active state: opacity 0.85

**Behavior:** On tap, navigate to `/library/add` (full-page form)

### Add Book Form (at `/library/add`)

**Page structure:**
- Header: "Add Book" (48px display)
- Form container: max-width 600px, centered, parchment background
- Form fields (all 52px height, 2px black border, Georgia font):
  - Title input (required)
  - Author input (required)
  - Cover Color picker (8 swatches, 56×56px each, arranged in 4 columns, 8px gap)
  - Notes textarea (optional, 4 rows, 52px base height expanding)
- Button group: "Save Book" (primary, aged gold) and "Cancel" (secondary, parchment) buttons side-by-side, 52px height, 2px borders

**Validation:**
- Title and Author are required (display error text below field if empty on submit)
- Color selection is required (visual feedback: selected swatch shows gold border)
- Notes are optional

**Behavior:**
- On "Save Book": POST new book to Supabase with `is_wishlist: false` and current user's `user_id`
- On save success: Immediately append book to local library state (optimistic update), then navigate back to `/library`
- On save failure: Display error state copy ("Failed to save. Please try again.") in toast or inline alert, remain on form
- On "Cancel": Navigate back to `/library` without saving

### Book Detail Page (at `/books/[id]`)

**Page structure:**
- Back button or breadcrumb: "← Library" (14px label weight, charcoal text, clickable)
- Book info container:
  - Title (48px display weight)
  - Author (14px label weight, text-secondary #666666)
  - Cover color swatch: 120×160px block rendered in book's `cover_color` value
  - Notes section (read-only, 14px body weight): "Notes: {notes}" or "No notes" if empty

**Styling:**
- Background: parchment (#F5F0E8)
- Borders: 2px black (#000000) around container and color swatch
- Text: Georgia serif, dark primary color (#222222)

**Behavior:**
- Fetch book data from Supabase using `[id]` dynamic segment
- Display all fields read-only (no edit capability in Phase 6)
- Back button navigates to `/library` (Phase 6 only from library; Phase 7 adds `/wishlist` path detection)

### Bookshelf Drag Reorder (Library page)

**Visual feedback:**
- **Ghost spine:** Semi-transparent copy (0.5 opacity) of the dragged book spine floats under cursor
- **Drop target gap:** Clear gap appears on the shelf where the spine will land; gap background highlighted in gold (#D4AF37)
- **Release:** Spine snaps into final position, gap closes

**Interaction:**
- dnd-kit SortableContext wraps the Bookshelf component
- Mouse, pointer, and touch sensors enabled
- Reorder updates local `books` state immediately (optimistic)
- Supabase `position` column updated in background (set all reordered books' positions)
- No loading state or confirmation needed (optimistic updates mask latency)

**Accessibility:**
- `role="listitem"` on each BookSpine
- `aria-label="{title}"` on each BookSpine
- Tab order follows visual order (shelf left-to-right, top-to-bottom)
- Drag indicator text for screen readers (e.g., "Use arrow keys to reorder, Enter to confirm")

## Interaction States

### Form Fields

- **Default:** Parchment background (#F5F0E8), 2px black border, Georgia serif, 16px body text
- **Focus:** 2px gold border (#D4AF37), parchment background
- **Error:** 2px red border (#8B1A1A), error text below field (13px, red)
- **Disabled:** Not applicable in Phase 6

### Buttons

- **Default:** Aged gold background (#D4AF37) for primary, parchment for secondary; 2px black border; 52px height; 16px body text
- **Hover:** Opacity 0.9 (slight darken)
- **Active:** Opacity 0.85 (pressed)
- **Disabled:** Opacity 0.5, cursor not-allowed (if validation fails)

### Book Spine (Draggable)

- **Default:** Original spine dimensions (56×150px), border 2px black, cursor pointer
- **Hover (desktop):** Border color changes to gold (#D4AF37)
- **Drag (in-progress):** Ghost copy floats at 0.5 opacity under cursor
- **Drop target:** Gold (#D4AF37) background highlight on gap position
