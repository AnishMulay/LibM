**Step A — library_screen.dart changes:**

Replace the `_chunk` / ShelfWidget rows approach in `_buildBody()` with a `ReorderableWrap` that:
1. Wraps ALL BookSpineWidget children in a single ReorderableWrap (reorderables handles visual row wrapping automatically)
2. Each child has `key: ValueKey(book.id)` so ReorderableWrap can track moves
3. Surrounds the ReorderableWrap in a Container that simulates the shelf aesthetic:
   - Parchment background, padding 16 all sides
   - Let ReorderableWrap handle the run spacing (runSpacing: 32 to give shelf-row visual separation)

`onReorder` handler — `_handleReorder(int oldIndex, int newIndex)`:
```dart
void _handleReorder(int oldIndex, int newIndex) {
  setState(() {
    final book = _books.removeAt(oldIndex);
    _books.insert(newIndex, book);
  });
  // Persist to Supabase asynchronously (don't await — optimistic UI)
  _bookService.updatePositions(_books.map((b) => b.id).toList());
}
```

Import to add at top of library_screen.dart:
```dart
import 'package:reorderables/reorderables.dart';
```

Updated `_buildBody()` when books are loaded (replace the chunk+ShelfWidget rows block):
```dart
return SingleChildScrollView(
  padding: const EdgeInsets.symmetric(horizontal: 16, vertical: 16),
  child: ReorderableWrap(
    spacing: 4,
    runSpacing: 32, // visual gap between shelf rows
    onReorder: _handleReorder,
    children: _books
        .map((b) => BookSpineWidget(key: ValueKey(b.id), book: b))
        .toList(),
  ),
);
```

Keep the empty state (ShelfWidget with isEmpty: true) and loading state unchanged.

Remove the `_chunk` method — it is no longer needed once ReorderableWrap handles wrapping.
